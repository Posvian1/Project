Определение стоимости автомобилей
Цель: Построить модель для быстрого определения рыночной стоимости автомобилей по техническим характеристикам.

Ключевые результаты:

Протестированы модели LinearRegression, RandomForest, LightGBM, CatBoost

Лучшая модель: CatBoostRegressor (баланс качества и скорости предсказания)

Рекомендована для использования в приложении "Не бит, не крашен"
